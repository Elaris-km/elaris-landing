# Elaris Landing

Текущая версия лендинга собрана на основе купленного статического шаблона `aigocy` и дорабатывается локально внутри этого репозитория.

## Команды

```bash
npm run verify
npm run build
npm run preview
```

После запуска `npm run preview` сайт доступен по адресу:

```text
http://127.0.0.1:4173
```

## Что проверяет `npm run verify`

- наличие основных файлов лендинга
- корректность локальных ссылок на ассеты в HTML
- подключение логотипов Elaris
- наличие `<title>Elaris</title>` в главной странице

## Примечание

Сейчас сайт работает как статический проект.

- `npm run preview` поднимает локальный сервер из корня проекта
- `npm run build` собирает deploy-версию в папку `dist/`
- для Vercel проект подготовлен под автосборку
